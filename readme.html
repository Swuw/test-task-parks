Installation test task from Andrei Yanushkevich

  1. Please import database with name "wp_plugins.sql" (located at the root folder of the project) to your local database server.
  2. Write in configuration file: wp-config.php Database settings.
  3. If for some reason this does not work, do not worry. It may not work on all web hosts. Open up wp-config-sample.php with a text editor like WordPad or similar and fill in your database connection details.
  4. User and Password for access to admin will send by email/messenger.
  5. Thank you.

System Requirements

  • PHP version 7.4 or greater.
  • MySQL version 5.0 or greater.

Recommendations

  • PHP version 7.4 or greater.
  • MySQL version 5.7 or greater OR MariaDB version 10.3 or greater.
  • The free and open-source cross-platform web server solution stack XAMPP, Open Server or other.
